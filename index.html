Markovian-tweets - A Twitter Bot Markov Sentence Generator

Overview

This project was designed to test our ability. We were asked to develop an application that would consume a text file containing sentences and produce a tweet, using a markov model, in the style of the input text. If we were able to complete this task and had we skills, then we were asked to add the ability of consuming a Twitter username, extracting a sample of that person's tweets, and generating a tweet in their style. Then, if we made it this far, we were challenged to create a bot which accepts a username and tweets a message in that user's style.
